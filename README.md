# Rabin-Karp Erlang

Stratus3D

Erlang implementation of the [Rabin-Karp string search algorithm](
https://en.wikipedia.org/wiki/Rabin%E2%80%93Karp_algorithm).

## API

`rabin_karp:search/2`
`rabin_karp:search/4`
